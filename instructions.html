Instructions

Introduction

As soon as the simulator is initiated, the address display is selected by default. You can input data into the address display using the hexadecimal keyboard keys. Upon entering an address, the dynamically stored data in that memory address is updated in the code display.

Entering the Keys

Clicking the * button allows you to switch to the other display for input. Use the INC button to increase the address immediately, regardless of which display is active. Similarly, use the DEC button to decrease the address, and you will see the stored data reflected in the code display.

Execution

After writing your complete code, do not forget to include the HALT instruction. Use the RESET button to reset all registers and flags, which will set the address display back to 0000. Initially, enter the starting address of your code, and then click the EXEC button to execute. The moment you press EXEC, it will appear in the code display.

Debugging Code

You can use the STEP button to execute statements one by one and observe the registers by clicking the Registers button below the simulator. You can also view the entire memory stored in the simulator by clicking on the Memory button.

In case the execution never halts, you will need to reload the page to initialize the simulator again. The good news is that the simulator does not lose entered data in memory. If you wish to clear the memory, use the Clear Memory button displayed in red alongside the memory data.
